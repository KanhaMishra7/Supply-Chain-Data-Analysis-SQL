### **Supply Chain Analytics: A Multi-Level SQL Deep Dive 📌**

### **Project Overview**

This project explores a comprehensive supply chain database featuring tables for **Suppliers**, **Products**, **Orders**, **Customers**, and **Order Items**. The analysis is structured into four distinct levels of complexity, moving from foundational data exploration to expert-level strategic market research.

### **🛠️ Tech Stack**

**Database:** SQL (Structured Query Language).

Techniques: Multi-table Joins, Aggregations, Common Table Expressions (CTEs), Window Functions, and Subqueries.
